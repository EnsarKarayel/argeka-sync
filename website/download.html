Teknik bilmeyen kullanici icin kurulum

  1. Setup dosyasini indirin. Yukaridaki Windows setup indir butonuna basin.
  2. Dosyayi calistirin. Indirilen ARGEKA-Sync-Setup.exe dosyasina cift tiklayin.
  3. Dili secin. 1 Turkce, 2 English.
  4. Bekleyin. Setup Git, Docker Desktop ve WSL kontrollerini yapar. Gerekirse sizi yonlendirir.
  5. Yeniden baslatma istenirse tekrar calistirin. Windows yeniden baslarsa ayni setup dosyasina tekrar cift tiklayin.
  6. Programi acin. Kurulum sonunda tarayici `http://localhost:8080` adresini acar. Sonraki acilislarda masaustundeki ARGEKA Sync.exe dosyasini kullanin.
