Kendi ortamında çalışan Windows kurulumu

Kurulum dosyasını indir, kendi bilgisayarında çalıştır.

ARGEKA Sync bizim sunucumuzda çalışmaz. Kurulum dosyasını indirirsiniz, kendi Windows bilgisayarınıza veya Windows sunucunuza kurarsınız. Veritabanı bağlantıları ve veriler sizde kalır.

Windows kurulum dosyası

Önerilen dosya ikonlu Windows kurulum dosyasıdır. Açıldığında Türkçe veya İngilizce kurulum seçilebilir.

Windows kurulum dosyasını indir CMD yedek kurulum

Google Ads ve kullanıcı güvenliği için bu sayfa otomatik indirme başlatmaz. Kurulum dosyası yalnızca kullanıcı butona bastığında indirilir.

Windows güvenlik uyarısı gösterirse dosya henüz imzalı olmadığı içindir. Kurumsal dağıtım için sonraki adım imzalı kurulum dosyası hazırlamaktır.

Kurulumdan sonra web sitesine veya uygulamaya kullanıcı adı ve şifre girmeniz gerekmez. Yerel çalışma alanı otomatik açılır.

Teknik bilmeyen kullanıcı için kurulum

  1. Kurulum dosyasını indirin. Yukaridaki Windows kurulum dosyasını indir butonuna basın.
  2. Dosyayı çalıştırın. İndirilen ARGEKA-Sync-Setup.exe dosyasına çift tıklayın.
  3. Dili seçin. 1 Türkçe, 2 English.
  4. Bekleyin. Kurulum Git, Docker Desktop ve WSL kontrollerini yapar. Gerekirse sizi yönlendirir.
  5. Yeniden başlatma istenirse tekrar çalıştırın. Windows yeniden başlarsa aynı kurulum dosyasına tekrar çift tıklayın.
  6. Programı açın. Kurulum sonunda tarayıcı `http://localhost:8080` adresini açar. Sonraki açılışlarda masaüstündeki ARGEKA Sync.exe dosyasını kullanın.
  7. Şifre beklemeyin. Panel yerel oturumu otomatik açar; sadece servis hazır değilse tekrar dene ekranı görünür.
