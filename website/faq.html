Sık sorulan sorular

Kurulum, veri güvenliği ve kullanım hakkında kısa cevaplar.

ARGEKA Sync bulut servis mi?

Hayır. Web sitesi sadece ürünü tanıtır ve kurulum dosyasını indirmenizi sağlar. Uygulama kullanıcının kendi Windows bilgisayarında veya sunucusunda çalışır.

Veritabanı şifreleri nereye gider?

Bağlantı bilgileri kurulum yapılan yerel ortamda tutulur. ARGEKA Sync bu bilgileri merkezi bir SaaS sistemine göndermez.

Hangi veritabanları aktif?

PostgreSQL/Internal, Microsoft SQL Server ve MySQL/MariaDB aktif çalışma motorlarıdır. Oracle, ODBC, CSV/Excel, REST API ve SAP HANA mimaride hazırlanan sonraki bağlantı tipleridir.

Kurulumdan sonra hesap gerekiyor mu?

Normal yerel kurulumda web hesabına giriş gerekmez. Panel localhost üzerinden yerel oturumu otomatik açar.

İndirme otomatik başlar mı?

Hayır. Google Ads ve kullanıcı güvenliği için sayfa açıldığında otomatik indirme başlamaz. Kurulum dosyası yalnızca kullanıcı indirme butonuna bastığında iner.

Canlı veritabanına bağlamadan önce ne yapmalıyım?

Önce test tablo veya test veritabanı ile deneyin, hedef verinin yedeğini alın ve kolon eşleme kurallarını küçük bir veri setiyle doğrulayın.
