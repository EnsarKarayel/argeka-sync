Sik sorulan sorular

Kurulum, veri guvenligi ve kullanim hakkinda kisa cevaplar.

ARGEKA Sync bulut servis mi?

Hayir. Web sitesi sadece urunu tanitir ve setup dosyasini indirmenizi saglar. Uygulama kullanicinin kendi Windows bilgisayarinda veya sunucusunda calisir.

Veritabani sifreleri nereye gider?

Baglanti bilgileri kurulum yapilan yerel ortamda tutulur. ARGEKA Sync bu bilgileri merkezi bir SaaS sistemine gondermez.

Hangi veritabanlari aktif?

PostgreSQL/Internal, Microsoft SQL Server ve MySQL/MariaDB aktif calisma motorlaridir. Oracle, ODBC, CSV/Excel, REST API ve SAP HANA mimaride hazirlanan sonraki baglanti tipleridir.

Kurulumdan sonra hesap gerekiyor mu?

Normal yerel kurulumda web hesabina giris gerekmez. Panel localhost uzerinden yerel oturumu otomatik acar.

Indirme otomatik baslar mi?

Hayir. Google Ads ve kullanici guvenligi icin sayfa acildiginda otomatik indirme baslamaz. Setup dosyasi yalnizca kullanici indirme butonuna bastiginda iner.

Canli veritabanina baglamadan once ne yapmaliyim?

Once test tablo veya test veritabani ile deneyin, hedef verinin yedegini alin ve kolon esleme kurallarini kucuk bir veri setiyle dogrulayin.
