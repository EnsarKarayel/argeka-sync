Neden ARGEKA Sync?

Bulut ETL karmaşası olmadan veritabanı aktarımı.

Veri sizin ortamınızda

Uygulama Docker üzerinde kendi bilgisayarınızda veya sunucunuzda çalışır. Veritabanı şifreleri üçüncü taraf bir SaaS ortamına taşınmaz.

Teknik ekibe uygun

SQL sorgusu, JSON parametre, kolon eşleme, hata politikası ve zamanlama ekranları sade ama doğrudan üretim ihtiyacına dönüktür.

Kurulum akışı hazır

Windows kurulum dosyası Git, Docker Desktop, WSL ve servisleri kontrol eder; uygulamayı yerelde başlatır ve masaüstüne ARGEKA Sync.exe ekler.

Çalışma akışı

Kur, bağlan, zamanla.

1

Bağlantıları ekleyin

Kaynak ve hedef veritabanı bilgilerini yerel panelde tanımlayın.

2

Sorgu ve kolonları eşleyin

Parametreli SQL sorgusunu, hedef tabloyu ve kolon davranışını belirleyin.

3

Zamanlayıp izleyin

Manuel veya periyodik çalıştırın; sonuç, hata ve satır sayılarını takip edin.

Farkımız

Teknik ekip için hafif, indirilebilir ve şeffaf.

Hesap zorunluluğu yok

Desteklenen motorlar

PostgreSQL/Internal, Microsoft SQL Server ve MySQL/MariaDB aktif çalışma motorlarıdır. Oracle, SQLite, ODBC, CSV/Excel, REST API ve SAP HANA connector mimarisine alınmıştır.
