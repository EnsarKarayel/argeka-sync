After installation, the app runs on the local computer by default: http://localhost:8080

First open

After installation, you do not need to type a username or password. The local workspace opens automatically.

Setup

The Windows setup checks Git, Docker Desktop and WSL, then adds ARGEKA Sync.exe to the desktop. If Windows asks for a restart, run setup again.

Backup

The backup.ps1 and restore.ps1 files in the project folder are prepared for database backup and restore.
