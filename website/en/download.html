Self-hosted Windows setup

Download the setup file and run it on your own computer.

ARGEKA Sync does not run on our servers. You download the setup file and install it on your Windows computer or Windows server. Database connections and data stay in your environment.

Windows Setup

The recommended file is the Windows setup executable. When opened, it lets you choose Turkish or English setup.

Download Windows setup Backup CMD setup

For Google Ads and user safety, this page never starts an automatic download. The setup file is downloaded only after the user clicks the button.

Windows may show a warning because the file is not code-signed yet. The next enterprise distribution step is a signed installer.

What gets installed?

  • ARGEKA Sync web panel
  • API service
  • PostgreSQL database
  • Docker services
  • Desktop ARGEKA Sync.exe launcher

Installation for non-technical users

  1. Download the setup file. Click the Download Windows setup button above.
  2. Run the file. Double-click ARGEKA-Sync-Setup.exe.
  3. Choose language. 1 Turkish, 2 English.
  4. Wait. Setup checks Git, Docker Desktop and WSL. It guides you when something is missing.
  5. If Windows asks for restart, run setup again. Restart the computer, then double-click the same setup file again.
  6. Open the program. Setup opens http://localhost:8080. Later, use ARGEKA Sync.exe on the desktop.
