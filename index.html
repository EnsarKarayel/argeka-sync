      Kurulum seçenekleri Roadmap
      1. Docker self-hosted: Nginx, API ve PostgreSQL ile tek komut kurulum.
      2. Windows installer: Tauri/Electron kabuğu, SQLite veya local API.
      3. Cloud: Yönetilen PostgreSQL, yedekleme, SSL, mail/takvim OAuth.
      4. Lisans: Online aktivasyon, offline lisans dosyası ve yıllık bakım.
      Maliyet stratejisi Kontrol
      • Cloud müşteriler aylık gelir ve merkezi operasyon sağlar.
      • Self-hosted müşteriler altyapı maliyetini kendi üstlenir.
      • Desktop paket destek yükü düşük, giriş ürünü olarak çalışır.
      • Enterprise kurulumlar entegrasyon ve destek geliri üretir.
