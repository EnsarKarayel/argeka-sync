# ARGEKA CRM Urun Paketleri

## Cloud Pro

- Hedef: hizli baslamak isteyen kucuk ve orta ekipler
- Model: aylik abonelik
- Altyapi: bizim yonettigimiz cloud
- Avantaj: kurulum yok, hizli onboarding

Onerilen fiyat: 749 TL / ay / takim baslangic paketi.

## Self-hosted Server

- Hedef: KVKK, veri gizliligi ve ic ag isteyen firmalar
- Model: yillik lisans + bakim
- Altyapi: musterinin VPS veya kurum sunucusu
- Avantaj: dusuk merkezi maliyet, yuksek kurumsal guven

Onerilen fiyat: 24.900 TL / yil baslangic.

## Desktop Starter

- Hedef: tek kullanici, danisman, kucuk ofis
- Model: tek seferlik lisans + istege bagli bakim
- Altyapi: Windows cihaz
- Avantaj: satin al, kur, kullan

Onerilen fiyat: 9.900 TL tek sefer.

## Enterprise

- Hedef: ERP, SAP, Odoo veya ozel entegrasyon isteyen firmalar
- Model: kurulum + lisans + destek
- Altyapi: musteri veya hibrit
- Avantaj: yuksek hizmet geliri ve uzun sureli sozlesme

Onerilen fiyat: proje bazli.

## Ticari karar

Ilk satis icin self-hosted ve desktop daha az altyapi maliyeti tasir. Cloud surum marka guveni ve surekli gelir icin tutulmali, fakat ilk kurumsal musteri self-hosted paketle kapatilabilir.
